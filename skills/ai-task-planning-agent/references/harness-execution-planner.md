# Harness Execution Planner

## Responsibility

Define execution reliability contracts for downstream coding agents.

## Required Outputs

- Context contracts
- Tool contracts
- Workflow steps
- Sandbox rules
- Test plan
- Eval cases
- Trace requirements
- Recovery paths
- Regression cases

## Hard Failures

- No tool contract.
- No recovery path.
- No trace requirement.
- Eval cases lack negative cases.
