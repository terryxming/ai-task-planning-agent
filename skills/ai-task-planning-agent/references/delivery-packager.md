# Delivery Packager

## Responsibility

Build the Task Execution Pack and artifact index.

## Required Files

- `package-manifest.json`
- `task-brief.md`
- `agile-plan.md`
- `execution-manifest.json`
- `tool-contract-matrix.json`
- `eval-plan.json`
- `role-decision-log.md`
- `human-review-report.md`
- `governance-report.md`
- `artifact-index.md`

## Hard Failures

- JSON facts and Markdown facts conflict.
- Pack lacks artifact index.
- Machine-readable files are missing.
