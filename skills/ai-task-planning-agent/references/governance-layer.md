# Governance Layer

## 职责

治理 schemas、manuals、fixtures、scripts、tests、release checklist、兼容性、rollback notes 和 `SKILL.md` 入口契约。

## 不负责

- 不自动修改用户 Skill。
- 不连接生产环境。
- 不替代项目管理工具。

## 必需输出

- Governance report
- Compatibility impact
- Release checklist status
- Regression fixture recommendations
- SKILL.md governance status
