# Role Council / Decision Log

## 职责

把角色责任绑定到输出字段，并记录关键决策。

## 常设角色

- Product owner
- Agile planner
- Harness engineer
- Agent architect
- Quality auditor
- Governance owner
- Opposition reviewer

## Decision log 字段

- Decision id
- Topic
- Proposal
- Challenge
- Evidence
- Decision
- Impacted artifacts
- Regression risk

## 硬失败

- 角色只发表评论，没有负责字段。
- 重大决策没有 challenge。
- 决策没有 impacted artifact。
