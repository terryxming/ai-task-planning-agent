# Role Council / Decision Log

## Responsibility

Bind roles to output responsibility and record decisions.

## Standing Roles

- Product owner
- Agile planner
- Harness engineer
- Agent architect
- Quality auditor
- Governance owner
- Opposition reviewer

## Decision Log Fields

- Decision id
- Topic
- Proposal
- Challenge
- Evidence
- Decision
- Impacted artifacts
- Regression risk

## Hard Failures

- Roles only comment without owning fields.
- No challenge recorded for major decisions.
- Decision has no impacted artifact.
